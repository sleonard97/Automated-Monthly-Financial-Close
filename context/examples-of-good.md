# Examples of Good

This file should be updated after the first generated output workbook exists.

## Good Flagged Row

Required fields:

- Account number
- Account name
- Account type
- YTD actual
- Current forecast
- Remaining dollars
- Percent forecast used
- Month-based benchmark
- Favorable/unfavorable label
- Flag reason
- AI review prompt

## Good AI Review Prompt

`Review because this account has used 83% of the full-year forecast by February, above the 67% straight-line benchmark. Confirm whether remaining forecast is sufficient.`

## Good Problems Entry

`Account 1234 appears in GL actuals but was not found in the financial model forecast detail. Review mapping before relying on variance status for this account.`
